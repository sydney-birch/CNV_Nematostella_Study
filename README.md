# CNV of Immune genes in Nematostella - Overview
This repository contains the primer generation, qPCR set up, and R analysis of the work looking at CNV of immune genes in Nematostella. 

## Study Info 
This qPCR study examines CNV of Immune genes across Nematostella populations. This study was conducted to complement the genomic variation investigation which sequenced four individuals from four populations (FL, ME, NC, NS) and identified CNV in immunity genes involved in the TRIM56 antiviral pathway. Here We generated primers using primer 3 for TRIM, STING, CGAS, and Catalyse (our control - 1 copy). We conducted qPCR with 3 biological replicates for these primers across 6 Nematostella populations: NS, ME, NH, MA, NC, and FL. 

## Overview of Pipeline: 
1. Generate primers for TRM, STING, CGAS, and Catalyse
2. Calculate primer efficency 
3. Design qPCR experiment and Run
4. Analyze qPCR Results
